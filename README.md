# OnRecord – Landing Page

Frontend landing page for **OnRecord**, an AI-based attendance system using  
**Face Recognition + Voice Recognition**.

This repository contains the public-facing website built with **Flask**, designed to present the product and redirect users to the live application.

---

## 🌐 Live Links

- **Landing Page** → https://onrecord-landingpage.vercel.app  
- **Main Application** → https://onrecord.streamlit.app  

---

## 🎯 Purpose

This landing page serves as:
- Product overview and feature showcase
- Entry point for users to access the application
- Structured presentation of system capabilities

---

## 🧩 Features

- Clean, responsive UI
- Structured product sections (features, workflow, benefits)
- Direct navigation to live AI application
- Lightweight Flask-based rendering
- Static asset management (HTML, CSS, components)

---

## 🛠️ Tech Stack

- **Flask** – lightweight backend for routing/templates  
- **HTML5 + CSS3** – UI structure and styling  
- **Jinja2** – template rendering
